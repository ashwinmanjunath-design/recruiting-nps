# Assets Folder

Place your logo image file here.

## Logo File

Please add your logo image file (PNG, SVG, or JPG) and name it `logo.png` (or update the import path in `Logo.tsx`).

Supported formats:
- `logo.png`
- `logo.svg`
- `logo.jpg` or `logo.jpeg`

Once you add the logo file, update the import in `client/src/components/Logo.tsx` to match your file name and format.
